# 🧠 Project Overview

Marketing teams frequently run experiments to determine whether advertisements increase user conversions.

This project evaluates the effectiveness of an advertising campaign using A/B testing and statistical hypothesis testing.

The analysis compares users exposed to advertisements (Ad Group) against users shown public service announcements (PSA Group) to determine whether advertising significantly improves conversion rates.

The project combines data validation, exploratory analysis, statistical testing, confidence interval estimation, power analysis, and business recommendations.

---

# 🏆 Key Results

✅ Analyzed 588,101 users

✅ Advertisement conversion rate: 2.555%

✅ PSA conversion rate: 1.785%

✅ Relative conversion lift: 43.1%

✅ Statistically significant improvement (p < 0.001)

✅ Identified high-exposure users as the primary driver of conversion uplift

# 🧪 Statistical Testing

### Two-Proportion Z-Test

| Metric | Value |
|----------|----------:|
| Z Statistic | 7.3701 |
| P-Value | 1.705e-13 |

### Decision

Reject the Null Hypothesis.

The advertisement campaign significantly improves conversion performance.

---

# 📏 Confidence Interval

95% Confidence Interval for Conversion Lift:

| Lower Bound | Upper Bound |
|----------|----------:|
| 0.595% | 0.943% |

### Interpretation

The true conversion uplift generated by advertising is estimated to be between 0.595% and 0.943%.
